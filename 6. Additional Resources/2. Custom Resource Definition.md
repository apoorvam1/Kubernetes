CRDs allow exntending the k8s APIs to create new resource objects. 


### Writing a CRD - Sample
```
apiVersion: apiextensions.k8s.io/v1beta1
kind: CustomResourceDefinition
metadata: 
  # name of the CRD is data and it'll be in the foo.bar domain
  name: database.foo.bar
spec:
  # group the CRD belongs to
  group: foo.bar
  version: v1
  scope: Namespaced
  names:
    plural: databases
    singluar: database
    kind: DataBase
    shortNames:
    - db
```
